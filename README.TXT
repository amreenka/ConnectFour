PROJECT TITLE: ConnectFour

PURPOSE OF PROJECT: The purpose of this project is to create a digital version of the classic ConnectFour game. The project aims to provide an interactive and engaging way for players to compete against each other, while also serving as a showcase of programming skills and game logic implementation.

VERSION or DATE: November 2022

HOW TO START THIS PROJECT: Open this project on Greenfoot and click "run"

AUTHORS: Amreen Kaur and Michelle Lee

USER INSTRUCTIONS: 
1 First player "red" uses the "A", "W", "D" keys to pick a spot for its token. Press "enter" when a spot has been picked.
2 Other player "blue"'s turn.
3 Repeat. The pieces stack from the bottom up.
4 The game ends when one player achieves a "Connect Four" or when the board is full, resulting in a draw.
